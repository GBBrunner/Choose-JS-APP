Setup Instructions

  1. Go to The Movie Database (TMDB).
  2. Sign up and obtain your API key under Settings ‚Üí API.
  3. Reference: Movie API Docs.

Example Endpoints:

  • Search by title: https://api.themoviedb.org/3/search/movie?api_key=YOUR_KEY&query=Harry%20Potter
  • Get movie by ID: https://api.themoviedb.org/3/movie/MovieIdHere?api_key=YOUR_KEY
